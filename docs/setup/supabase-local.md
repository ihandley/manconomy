# Supabase Local Development

## Start services
pnpm supabase:start

## Stop services
pnpm supabase:stop

## Reset database (runs seeds)
pnpm supabase:reset

## View status
pnpm supabase:status

## URLs
- API: http://localhost:54321
- Studio: http://localhost:54323

## Notes
- Requires Docker Desktop running
- Config is in `supabase/config.toml`
