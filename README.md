# Manconomy

Monorepo for the Manconomy project.

## Structure

- `apps/` - user-facing applications
- `packages/` - shared libraries, types, config, UI
- `services/` - backend services, workers, agents
- `scripts/` - local automation and maintenance scripts
- `docs/` - architecture, product, ADRs, and setup docs

## Getting Started

1. Install pnpm
2. Install dependencies
3. Run the app from `apps/web`
